Welcome to Announcements Website

Announcements is a Revolt Bot. Add it to your server today!

a!setannouncement use this to allow users to subscribe to this channel from their server to recieve announcements (Either add a channel id or use this command in the channel you wish to make an announcement channel)

a!subscribe channel-id replace the channel-id with the channel id you wish to subscribe to for announcements (It would be the channel id from another guild)

a!removeannouncement use this to remove a channel as an announcement channel (it works the same way as setannouncement)

a!unsubscribe channel-id use this to stop recieving announcemenets from a channel's announcement channel (works the same way as setannouncement just different function)
